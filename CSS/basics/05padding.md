# padding（内边距）
* padding-top
    * 上边内边距
* padding-right
    * 右边内边距
* padding-bottom
    * 下边内边距
* padding-left
    * 左边内边距
* padding:top right bottom left;
* 注意：内边距相当于给一个盒子加了填充厚度会影响盒子大小。
## padding复合写法：
* 只有一个属性值时：4个方向都是同一个值。
* 设置两个属性值时：
    * 第一个属性值设置的是上下两个方向
    * 第二个属性值设置的是左右两个方向
* 设置三个属性值时：
    * 第一个属性值设置的是上边一个方向
    * 第二个属性值设置的是左右两个方向
    * 第三个属性值设置的是下边一个方向
