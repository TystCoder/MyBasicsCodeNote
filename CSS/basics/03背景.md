# 背景
## 背景和内容的区别
* 内容是会撑开容器宽高
* 背景不会占用容器宽高
## background
* background-color——背景颜色
    * 颜色英文关键字
    * rgb
    * 十六进制
* bakcground-image:url();——背景图
    * 背景图默认铺满整个容器的大小
* background-repeat:;——背景图是否重复
    * no-repeat
        * 不重复
    * repeat-x
        * x轴重复
    * repeat-y
        * y轴重复
    * repeat
        * x,y轴重复
* bakcground-postiton:x y;——背景图位置
    * 具体数值：
        * left
        * right
        * top
        * bottom
        * centent
    * 当第二个属性值没有填写，默认居中
* bakcground-attachment:;——背景图是否滚动
    * fixed
        * 固定在浏览器可视区域
    * scroll
        * 跟随滚动条滚动
## background复合样式
* 不分属性书写顺序的
    * 颜色相关
    * 图片相关
    * 是否重复
    * 图片位置
    * 是否滚动
