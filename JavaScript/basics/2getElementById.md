# getElementById

```javascript
document.getElementById
```

* **.**&rArr;的
* getElementById
  * get&rArr;获取
  * Element&rArr;元素

## 调试

```javascript
console.log()
```

* 可以利用console.log(要打印的内容)，来此确认内容是否正确

## 特性

* 通过id进行获取
* 如果获取一个不存在的元素，得到的结果会是null！！！
* getElementById只能document下有,id是当前页面中唯一的
* 遇到第一个符合条件的元素，就返回第一个元素
